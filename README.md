# 在 VS Code 的终端中阅读电子书插件

该插件支持开发者在 VS Code 的终端中阅读电子书（更隐蔽的摸鱼方式）。

## 使用方法
1. 导入本地的电子书，支持多选导入
2. 点击阅读
